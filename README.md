# Study Buddy Chatbot 🤖📚

Study Buddy Chatbot is a web-based chatbot designed to help students with study-related queries through an interactive and user-friendly interface.

## Features
- Simple chatbot interface
- Interactive conversation system
- Clean and responsive design
- Built for students to assist with learning

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
index.html – Main webpage  
page.html – Chatbot interface  
welcome.html – Welcome screen  
style.css – Styling for the website  
script.js – Chatbot functionality  

## How to Run
1. Download the repository
2. Open `index.html` in a web browser
3. Start interacting with the chatbot
